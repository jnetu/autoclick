#  Auto Clicker +99999 CPS

Um Auto Clicker de alto desempenho desenvolvido em Python. Utiliza a biblioteca `pynput` e múltiplas threads (`threading`) para contornar gargalos de performance e entregar **15+ Cliques Por Segundo (CPS)**. 

Totalmente controlável via terminal, com monitoramento de CPS em tempo real.

##  Funcionalidades

- **Multi-threading:** Utiliza 6 threads paralelas para maximizar a velocidade dos cliques.
- **Monitor de CPS:** Acompanhe a velocidade real dos cliques direto no terminal.
- **Ativação Rápida:** Ligue e desligue instantaneamente usando o botão lateral do mouse (Botão X2 / Thumb).
- **Velocidade Customizável:** Altere o intervalo de cliques em tempo real, sem precisar reiniciar o programa.
- **Interface Colorida:** Terminal interativo e fácil de ler.

##  Como baixar e usar (Usuários Comuns)

Se você quer apenas usar o programa, não precisa instalar o Python!

1. Vá até a aba [Releases](../../releases) do projeto.
2. Baixe o arquivo `.exe` mais recente.
3. Dê um duplo clique no executável. O terminal abrirá automaticamente.
4. Siga as instruções na tela para configurar a velocidade e use o **botão lateral do mouse** para ativar/desativar.
